Angles Cheat Sheet

• **Acute:** $0^\circ < \text{Angle} < 90^\circ$
• **Right:** $\text{Angle} = 90^\circ$
• **Obtuse:** $90^\circ < \text{Angle} < 180^\circ$
• **Straight:** $\text{Angle} = 180^\circ$
• **Reflex:** $180^\circ < \text{Angle} < 360^\circ$
• **Complete:** $\text{Angle} = 360^\circ$
