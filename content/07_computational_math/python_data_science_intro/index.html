Homogeneous Arrays vs. Lists

In standard Python, lists are highly flexible container structures, but they are too slow for advanced math operations. Data science libraries bypass Python lists and use homogeneous arrays (like NumPy's ndarray). These arrays store data in contiguous blocks of memory, allowing operations to execute simultaneously across the entire array in a compiled C backend—a concept known as Vectorization. 

import numpy as np

# Python List Addition (Requires slow loop iteration)
list_a = [1, 2, 3]
list_b = [4, 5, 6]
list_sum = [x + y for x, y in zip(list_a, list_b)]

# NumPy Array Vectorized Addition (C-Speed contiguous execution)
arr_a = np.array([1, 2, 3])
arr_b = np.array([4, 5, 6])
arr_sum = arr_a + arr_b  # array([5, 7, 9])

The Python Scientific Stack

For computation, Python features a set of core libraries designed specifically to handle numerical analysis, statistics, and visualizations:

  • NumPy: Handlers for multidimensional arrays, vector algorithms, and basic linear algebra.
  • Pandas: Handles tabular structures like DataFrames (essentially programmatical spreadsheets) for grouping, filtering, and cleaning data.
  • SciPy: Scientific computations including numerical integrals, differential systems, and advanced statistics.
  • SymPy: Symbolic mathematics. Unlike NumPy which calculates values to decimals (e.g. 1.414), SymPy represents algebraic entities analytically (like keeping variables as x or constants as √2).
  • Matplotlib: The foundation plotting library used to draw curves, distributions, and scatter grids.

Practical Computational Math Recipes

Below are code recipes demonstrating how students solve common computational tasks:

Recipe 1: Vectorized Sine Wave Generator

Instead of running a loop to calculate angles one by one, we process an array of 1,000 points simultaneously:

import numpy as np
import matplotlib.pyplot as plt

# Generate 1000 steps from 0 to 2*pi
x = np.linspace(0, 2 * np.pi, 1000)
y = np.sin(x)  # Calculates sin values for all 1000 elements

plt.plot(x, y, color='#6366f1', linewidth=2)
plt.grid(True)
plt.show()

Recipe 2: Solving Linear Algebra Systems ($Ax = b$)

Solves for x and y in a system of equations: $2x + 3y = 8$, and $5x - y = 3$.

import numpy as np

# Coefficient Matrix A
A = np.array([
    [2,  3],
    [5, -1]
])

# Constants Vector b
b = np.array([8, 3])

solution = np.linalg.solve(A, b)
print(f"x = {solution[0]}, y = {solution[1]}")  # Outputs: x = 1.0, y = 2.0

Recipe 3: Monte Carlo Simulation (Approximating π)

Simulates throwing 100,000 random darts inside a $2 \times 2$ grid to approximate the mathematical constant π:

import numpy as np

n = 100000
x = np.random.uniform(-1, 1, n)
y = np.random.uniform(-1, 1, n)

# Pythagoras: distance from origin
inside = (x**2 + y**2) <= 1
pi_approx = 4 * np.sum(inside) / n

print(f"Approximated Pi: {pi_approx}")

Reference Cheat Sheet

A reference mapping mathematical notation to code equivalents in Python libraries:

Concept / Notation Library Code Syntax Description
Array / Vector np.array([1, 2, 3]) Homogeneous array structures.
Interval $[a, b]$ np.linspace(a, b, n) Generates $n$ evenly spaced points.
Scatter Matrix plt.scatter(x, y) Draws coordinate pairs $(x,y)$ on a grid.
Symbolic Variable $x$ sympy.Symbol('x') Initializes symbolic variable for algebraic equations.
Integral $\int_a^b f(x)dx$ scipy.integrate.quad(f, a, b) Computes numerical definite integration.
Note: When implementing code math, always check if your output arrays are contiguous to prevent memory leaks during massive iterative loops!
