Number Systems Hierarchy

Master standard number classifications. Analyze how integers, rational numbers, and real numbers nest inside each other in a mathematical set hierarchy.

Classification for 7

Set Descriptions

* Natural (ℕ): Counting numbers starting at 1.

* Whole (𝕎): Natural numbers plus zero (0).

* Integers (ℤ): Whole numbers plus negative integers.

* Rationals (ℚ): Fractional ratios (terminating/repeating decimals).

* Irrationals (ℙ): Non-repeating infinite decimals (like $\pi$ or $\sqrt{2}$).

* Reals (ℝ): The complete coordinate scale (Rationals + Irrationals).

Natural, Whole, and Integer sets sit nested inside each other. If a number is Natural, it is automatically Whole, Integer, and Rational!
