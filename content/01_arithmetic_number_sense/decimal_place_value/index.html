Base-10 Multiplier Rule

Moving **LEFT** multiplies the digit's value by $10$.

Moving **RIGHT** divides the digit's value by $10$ (or multiplies by $0.1$).

The decimal point separates the whole number portion (left) from the fractional decimal portion (right).
