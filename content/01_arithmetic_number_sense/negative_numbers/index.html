Directed Vector Operations

Adding positive numbers moves to the **right** (increasing). Adding negative numbers (or subtracting positive numbers) changes direction and moves to the **left** (decreasing).

Equation Output

0 + 0 = 0
Starting Point (Value A) -2
Operation
Offset/Vector (Value B) +5
Visual Vector Logic:
Starting at coordinate -2. Adding positive +5 draws a vector arrow pointing to the right, completing at final coordinate 3.
