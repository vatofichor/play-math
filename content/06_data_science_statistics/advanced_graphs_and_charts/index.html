What is a Histogram?

Unlike a bar chart representing distinct categories, a **Histogram** shows the frequency distribution of **continuous** values grouped into ranges (bins).

* **Touching Bars:** The bars touch each other because they cover continuous numeric boundaries (e.g. $[50, 60), [60, 70)$).

* **Bin Count:** Changing the **Bin Width** alters the shape of the graph, either detailing individual spikes or smoothing out the general trend.

Interactive Tip

Try moving the slider. If you make the bin width very small, you'll see gaps. If you make it too large, the details of the distribution merge into a single massive block!

Understanding Box Plots

A **Box Plot** (Box-and-Whisker) summarizes data using a five-number summary: Min, Q1, Median, Q3, Max.

* **The Box:** Represents the middle 50% of values, extending from the Lower Quartile ($Q_1$) to the Upper Quartile ($Q_3$).

* **The Whiskers:** Extend out to the lowest and highest values that are *not* outliers.

The 1.5 × IQR Outlier Rule

The Interquartile Range is calculated as: **$\text{IQR} = Q_3 - Q_1$**.

Any data value is mathematically defined as an **outlier** if it lies:
Below $Q_1 - 1.5 \times \text{IQR}$
Above $Q_3 + 1.5 \times \text{IQR}$
Outliers are displayed as red glowing star symbols ($\star$).

Interactive: Drag any dot horizontally in the top row to change its value. Watch how the box and outlier bounds react!

Calculating Circle Sectors

A **Pie Chart** represents categorical data relative to a total sum.

* **Percentage Share:** $$\text{Share} = \frac{\text{Category Value}}{\text{Total Sum}} \times 100\%$$

* **Central Angle:** Since a full circle contains $360^\circ$: $$\text{Angle} = \frac{\text{Category Value}}{\text{Total Sum}} \times 360^\circ$$

Interactive Tip

Hover over the circle slices to highlight them. The dynamic center values display the sector's name, absolute value, relative percentage, and exact central angle.

A pie chart is ideal for showing composition relative to a whole, but is less effective if there are too many small sectors.
