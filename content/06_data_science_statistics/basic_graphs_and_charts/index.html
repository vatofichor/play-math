Reading Bar Charts

* The **X-Axis (Horizontal)** displays the different categories.

* The **Y-Axis (Vertical)** shows numerical values (scale count).

* The **Height** of each bar matches its value. A taller bar means a larger quantity!

Interactive Tip

Click the **+** or **−** buttons to adjust values. Watch the chart scale adjust itself dynamically to display the new quantities.

A bar chart is best for comparing values across separate groups or items.

Reading Line Graphs

* **Intervals (X-Axis):** Represents consecutive points in time (e.g. Weeks, Days, Hours).

* **Values (Y-Axis):** The numeric measurements at each time point.

* **Trends:** Points are connected by a line. A rising line shows **growth/increase**, while a falling line shows a **decrease**!

Interactive Tip

Modify line graph values below the canvas. Hover over the glowing coordinate points to inspect exact readings.

Line graphs are ideal for showing how data shifts continuously over time.

Key Scale Concepts

A **Pictograph** uses pictures or symbols to represent data values.

The **Key** tells us how much value a single picture represents.

If a category has an odd value under a scale of 2, a **Half Symbol** is drawn to show a count of 1!

Pictograph Calculation

Symbols drawn = Value ÷ Key
Using a larger key value (e.g. 5 or 10) allows pictographs to easily display very large numbers without drawing too many icons.
