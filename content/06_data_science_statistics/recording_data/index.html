Pictograph Rules


  • Scale Key: A key specifies how many units each graphical symbol represents. Always multiply the count of icons by the scale key value to find the real amount.
  • Half-Icons: If the scale key is $K$, a half-drawn icon represents exactly $K / 2$ observations. E.g., if key = 4, a half-icon represents 2 units.
  • Ledger Compilation: Summing chronological logs systematically into categories is the foundational step of all data processing.
