Proof

Algebraic Expansion

Why does this identity always hold true? Let's prove it by distributing the binomial expression:

1. Start with the squared expression:
(a + b)² = (a + b)(a + b)

2. Apply the distributive law:
= a(a + b) + b(a + b)

3. Distribute variables into the terms:
= a² + ab + ba + b²

4. Since multiplication is commutative (ab = ba), combine the like terms:
= a² + 2ab + b²

Concrete Example: Let's check the identity with real numbers, evaluating the expression (a + b)² when a = 3 and b = 5:

1. Evaluate the left side directly by summing first:
(3 + 5)² = 8² = 64

2. Expand the right side using the identity:
a² + 2ab + b² = 3² + 2(3)(5) + 5²

3. Calculate individual terms:
= 9 + 30 + 25

4. Sum the expanded terms:
= 64
Both paths yield exactly 64, verifying the algebraic expansion holds true!
