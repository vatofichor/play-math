How to Factor a Trinomial: x² + Bx + C

  1. Identify the sum value B and the product constant C.
  2. List the integer factor pairs of C (numbers that multiply to C).
  3. Sum each pair to find which one equals B.
  4. Choose this matching pair (p, q) where: p + q = B and p · q = C.
  5. Write the factored expression in the form: (x + p)(x + q).

If we expand (x + 4)(x + 2), what is the linear coefficient of the resulting trinomial?

Correct Answer!

Expanding gives x² + 2x + 4x + 8 = x² + 6x + 8. The linear coefficient is 6 (which is 4 + 2).
