Standard Rules


  • Linear Standard Form: Ax + By = C
  • Constraints: A ≥ 0, and A, B, C are integers.
  • Quadratic Standard Form: ax² + bx + c = 0
  • Quadratic Vertex x-pos: h = -b / (2a)
